# blog

a [Sails](http://sailsjs.org) application



##GET
``
   /
  /post/home               VIEW post
  /admin/dashboard        
  /admin/edit/:id          EDIT post
  /post/show/:id           SHOW post
``

##UPDATE
``
  /admin/update/:id         UPDATE post as admin
``

##DELETE
``
  /admin/delete/:id         DELETE post as admin
``

##CREATE 
``
  /post/create        CREATE NEW post 
``
